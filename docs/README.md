# Intake docs

This directory is the GitHub Pages source for Intake.

Start with [`index.html`](index.html), then use these focused pages:

- [`APP_CAPABILITIES.md`](APP_CAPABILITIES.md) — app capability boundaries
- [`QA.md`](QA.md) — QA plan and acceptance checklist
- [`OPERATIONS.md`](OPERATIONS.md) — local run and operations guide
- [`THREAT_MODEL.md`](THREAT_MODEL.md) — threat model and safety boundary
- [`PACKAGING.md`](PACKAGING.md) — GHCR package publishing
- [`RELEASE.md`](RELEASE.md) — release workflow and tagging process

The Pages workflow publishes this directory when GitHub Pages is configured to use GitHub Actions.
