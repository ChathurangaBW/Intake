# Intake docs

This directory is the GitHub Pages source for Intake.

- [`RELEASE_NOTES_1.0.0.md`](RELEASE_NOTES_1.0.0.md) — final 1.0 release summary
- [`FINAL_PRODUCT_CHECKLIST.md`](FINAL_PRODUCT_CHECKLIST.md) — final product readiness checklist
- [`DEMO.md`](DEMO.md) — demo workflow and seed data
- [`SDK.md`](SDK.md) — Python SDK usage
- [`QA.md`](QA.md) — QA plan and acceptance checklist
- [`OPERATIONS.md`](OPERATIONS.md) — local run and operations guide
- [`THREAT_MODEL.md`](THREAT_MODEL.md) — threat model and safety model
- [`SECURITY_BOUNDARY.md`](SECURITY_BOUNDARY.md) — explicit allowed and blocked runtime behaviors
- [`SUPPLY_CHAIN.md`](SUPPLY_CHAIN.md) — SBOM, provenance, scanning, and artifact verification
- [`PACKAGING.md`](PACKAGING.md) — GHCR package publishing
- [`CONTAINER.md`](CONTAINER.md) — Compose and published-container usage
- [`RELEASE.md`](RELEASE.md) — release workflow and tagging process
- [`BADGES.md`](BADGES.md) — README badge behavior
